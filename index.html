ldParallax

parallax in both pure css and js

About

ldParallax is a css / js library for creating parallax effects in your web page.

Usage

include ldpx.css in your website:

<link rel="stylesheet" type="text/css" href="ldpx.css"/>

add an element with ldpx and ldpx-full class as the topmost element:

<div class="ldpx ldpx-full"> <!-- all your content inside here --> </div>

for any element you'd like to do parallax, add a ldpxl class with an optional ds-* class, * from 1 ~ 8 for the amount of lift:

<div class="ldpx ldpx-full"> <div class="ldpxl zs-1" style="background:#f00"> A </div> <div class="ldpxl zs-2" style="background:#f90"> B </div> <div class="ldpxl zs-3" style="background:#ff0"> C </div> <div class="ldpxl zs-4" style="background:#9f0"> D </div> <div class="ldpxl zs-5" style="background:#0f0"> E </div> </div>

As a container

Instead of parallax when scrolling the web page, you can also do parallax in a scrollable container. Similarly, setup a container with ldpx but without ldpx-full class, and layers in ldpxl classes. Note: you probably will have to do some extra styling to make the container scrollable:

<div class="ldpx" style="height:200px;padding:200px 0 200px"> <div class="ldpxl zs-1" style="background:#f00"> A </div> <div class="ldpxl zs-2" style="background:#f90"> B </div> <div class="ldpxl zs-3" style="background:#ff0"> C </div> <div class="ldpxl zs-4" style="background:#9f0"> D </div> <div class="ldpxl zs-5" style="background:#0f0"> E </div> </div>

External Scrolling Element

You can use JS part of ldParallax to get the parallax effects using differet scrolling element outside the element with ldpx class. This will be documented later.

To initialize with JS, passing the scrolling element as the host option to the ldParallax constructor:

var el = document.querySelector("your-scrolling-container"); var ldpx = new ldParallax({host: el});

All nodes with ldpx class under the host element will be added to this object automatically.

To add nodes outside the scrolling element is under construction.

Compatibility

CSS part of ldParallax uses following modern web technologies:

  • CSS 3D Transform

CSS 3D Transform is not supported by IE <= 9, so please don't use ldParallax if you need IE <= 9 support.

Furthermore, JS part of ldParallax uses following technologies that IE ( all versions ) don't support but it's possible to find polyfill for that.

  • Array.from, Array.isArray
  • IntersectionObserver
  • document.scrollingElement
